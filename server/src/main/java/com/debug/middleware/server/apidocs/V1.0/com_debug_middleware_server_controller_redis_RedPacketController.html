请求URL

/red/packet/hand/out POST

{
	"userId":"int //用户id",
	"total":"int //红包个数【必须】",
	"amount":"int //总金额,单位为分【必须】"
}

返回结果

{
	"code":"int //状态码",
	"msg":"string //描述信息",
	"data":{}
}
